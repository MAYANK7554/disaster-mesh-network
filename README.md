## Project Overview
This project demonstrates the concept and implementation of a Disaster-Resilient Mesh Communication Network. It showcases how mesh topologies can autonomously reroute data traffic when a critical infrastructure node fails, ensuring continuous communication during disasters.

The project features a theoretical network design built in **Cisco Packet Tracer** and a programmatic, graph-based simulation built in **Python**.

## Objectives
- To design a decentralized mesh network topology without a single point of failure.
- To simulate network traffic routing using shortest-path algorithms.
- To demonstrate network resilience by dynamically rerouting traffic after an unexpected node failure.

## 🛠️ Tools & Technologies Used
* **Cisco Packet Tracer:** For designing the physical and logical network topology.
* **Python 3:** For algorithmic simulation.
* **NetworkX:** Python library for the creation and study of complex networks.
* **Matplotlib:** For visualizing the network nodes and routing paths.
